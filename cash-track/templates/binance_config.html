⚙️ Configuración de Binance API

ℹ️ Cómo obtener tus API Keys de Binance

Opción 1: Desde el sitio web (recomendado)

  1. Inicia sesión en Binance.com
  2. En la esquina superior derecha, pasa el cursor sobre tu icono de perfil
  3. En el menú que aparece, busca y haz clic en "API Management"
  4. Si no ves "API Management", ve directamente a: Configuración de API
  5. Haz clic en "Create API" y selecciona "System generated"
  6. Completa la verificación de seguridad (2FA, email, etc.)
  7. Ingresa un nombre para tu API (ejemplo: "AurefiQ")
  8. MUY IMPORTANTE: En los permisos, marca SOLO:
    • ✅ "Enable Reading" (Habilitar lectura)
    • ❌ NO marques "Enable Spot & Margin Trading"
    • ❌ NO marques "Enable Withdrawals"
    • ❌ NO marques "Enable Futures"
  9. Copia tu API Key y Secret Key (guárdalos en un lugar seguro)

Opción 2: Desde la app móvil

  1. Abre la app de Binance
  2. Toca tu icono de perfil (abajo a la derecha)
  3. Ve a Configuración → API Management
  4. Sigue los pasos 5-9 de la opción anterior

⚠️ Seguridad: Nunca compartas tus API Keys. Solo se usarán para leer balances y precios, NO para realizar transacciones, trading o retiros.

💡 Nota: Si no encuentras "API Management" en el menú de tu perfil, es posible que necesites acceder desde el navegador web de escritorio o usar el link directo proporcionado arriba.
